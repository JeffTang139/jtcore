本包下放置逻辑框架下所引发的异常。
规则如下:
1. 所有逻辑框架定义的异常必须继承自CoreException，并放置在本包下。
2. 所有自定义异常的消息文本都必须参数化，即创建异常时仅指定参数，而不是文本。文本在异常类的构造方法中格式化。
3. 逐步地重构旧有的引发异常的代码，用新的异常替换旧的异常。
4. 所有的自定义异常在本文本中适当描述用途，本文本随着异常的增多而变化，常用的异常的描述靠前，不常用的靠后。
